# PlaylistForge 0.17.0-alpha70 — Public Alpha

This is the first public tester package distributed through the official PlaylistForge Downloads repository.

## Highlights

- Embedded Watch mode for Live TV, movies and TV shows
- Saved-project switcher without leaving Watch mode
- Poster libraries, seasons, episode history and provider metadata
- Playback seeking, volume, audio tracks and subtitles
- TiviMate-style channel OSD with EPG now/next information
- Separate Live TV, Movies and TV Shows libraries in Watch and Editor
- Large-playlist responsiveness improvements
- Two-column Settings layout and responsive application shutdown

## Requirements

- Windows 10 or Windows 11, x64
- No separate .NET installation required
- Extract the complete ZIP before starting `PlaylistForge.exe`
- Keep the included `libvlc` folder beside the executable

## Public alpha notice

This build is intended for testing and feedback. It is not code-signed yet, so Windows SmartScreen may display an unknown-publisher warning.

PlaylistForge does not provide channels, playlists, subscriptions or media content.
