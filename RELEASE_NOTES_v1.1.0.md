# PlaylistForge 1.1.0 — Public Beta

## Highlights

- **Grouped Movies and TV Shows editor library.** The editor now mirrors Watch mode: movies are poster cards, and all episodes of a TV show are collected under one series card instead of appearing as a scattered flat list.
- **Edit without losing tools.** Click a card to open the existing full editor grid for that movie or only that show’s episodes. All existing F2/double-click editing, locking, hiding, EPG and logo actions remain available.
- **Natural episode order.** TV entries are consistently ordered by show, season and episode, so `Bodyguard S01/S02`, `Cobra Kai`, and similar provider names stay together.
- **Responsive large catalogs.** Posters load lazily only around the visible viewport, with four concurrent image requests.

## Requirements

- Windows 10 or Windows 11, x64
- No separate .NET installation required
- Run `PlaylistForge-Setup-1.1.0.exe` — no manual ZIP extraction needed
- Installing over an existing install upgrades it in place

## Public beta notice

This build is intended for testing and feedback. It is not code-signed yet, so Windows SmartScreen may display an unknown-publisher warning on both the installer and the app itself.

PlaylistForge does not provide channels, playlists, subscriptions or media content.
